سلام دنیا!

برای برنامه نویسان، عبارت «سلام دنیا!» مشابه یک آیین سنتی است. این کد کوتاه، یک پیام ساده ولی بسیار پر اهمیت با خود دارد. اول آنکه نشان می‌دهد همه چیز به درستی کار می‌کند. دوم آنکه فرد می‌تواند خود مستقیم با کامپیوتر صحبت کند. این اولین قدم در مسیر ایجاد برنامه‌های پیچیده‌تر است. برای بسیاری از مبتدیان، لحظه‌ای که عبارت «سلام دنیا!» را بر روی نمایشگر خود می‌بینند نشانه ملموسی است که ماجراجویی کد نویسی‌شان آغاز شده است.

print("سلام دنیا!")

تابع سینوسی با فرکانس‌های مختلف

این نمودار به شما این امکان را می‌دهد تا توابع سینوسی با فرکانس‌های مختلف را بررسی کنید. می‌توانید فرکانس را با استفاده از اسلایدر بالای نمودار بین مقادیر ۱ تا ۱۰ تنظیم کنید. 

  
# Import matplotlib and numpy
import matplotlib.pyplot as plt
import numpy as np
# Import ipywidgets for the slider and the interact function
import ipywidgets as widgets
from ipywidgets import interact
# Define a function to plot the sine function with a given frequency
def plot_sine(frequency):
  # Create an array of x values from 0 to 2 pi
  x = np.linspace(0, 2 * np.pi, 100)
  # Compute the sine of x with the given frequency
  y = np.sin(frequency * x)
  # Plot the sine curve
  plt.plot(x, y)
  # Show the plot
  plt.show()
# Create a slider widget for the frequency
slider = widgets.FloatSlider(min=1, max=10, step=0.1, value=1)
# Use the interact function to link the slider and the plot_sine function
interact(plot_sine, frequency=slider)
