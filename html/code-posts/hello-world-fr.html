Salut le monde!

Pour les programmeurs, le "Salut le monde!" est un rite de passage. Ce court extrait de code affiche un message simple, mais son importance est profonde. Il vérifie la réussite de l'installation, prouvant que le programmeur peut écrire du code compris par l'ordinateur. Plus important encore, c'est le premier pas vers la création de programmes complexes. Pour de nombreux débutants, c'est un moment de triomphe, un signe tangible que leur aventure de codage a commencé.

Fonction sinusoidale avec différentes fréquences

Cette visualisation vous permet d'explorer des fonctions sinusoidales avec différentes fréquences. Vous pouvez ajuster la fréquence à l'aide du curseur interactif au-dessus du graphique. Le curseur vous permet de choisir des valeurs entre 1 et 10.

    
# Import matplotlib and numpy
import matplotlib.pyplot as plt
import numpy as np
# Import ipywidgets for the slider and the interact function
import ipywidgets as widgets
from ipywidgets import interact
# Define a function to plot the sine function with a given frequency
def plot_sine(frequency):
    # Create an array of x values from 0 to 2 pi
    x = np.linspace(0, 2 * np.pi, 100)
    # Compute the sine of x with the given frequency
    y = np.sin(frequency * x)
    # Plot the sine curve
    plt.plot(x, y)
    # Show the plot
    plt.show()
# Create a slider widget for the frequency
slider = widgets.FloatSlider(min=1, max=10, step=0.1, value=1)
# Use the interact function to link the slider and the plot_sine function
interact(plot_sine, frequency=slider)
